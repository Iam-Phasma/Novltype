# Novltype

Novltype is a minimal, keyboard-first typing trainer focused on clean visuals, responsive input, and mechanical-style key audio.

## Live Demo

https://iam-phasma.github.io/Novltype/

## Features

- Three display modes: single word, scrolling paragraph, and rise mode
- Practice filters by keyboard row and word length
- Timed sessions: free, 30s, 1m, 2m, 3m, 5m
- Jump mode and strict key mode
- Live metrics for time, WPM, accuracy, and words
- Theme toggle with system preference support
- Key press and release sounds with low-latency warmup
- Progressive Web App support with install prompt capability

## Tech Stack

- Vite 6
- Tailwind CSS 4 (plus custom CSS)
- Vanilla ES modules for game logic
- Small TypeScript adoption for shared DOM references
- vite-plugin-pwa for manifest and service worker generation

## Local Development

Requirements:
- Node.js 20+

Install dependencies:

npm install

Run dev server:

npm run dev

Build for production:

npm run build

Preview production build locally:

npm run preview

## GitHub Pages Deployment

This project deploys through GitHub Actions.

1. Ensure GitHub Pages source is set to GitHub Actions in repository settings.
2. Push to main.
3. Workflow at .github/workflows/deploy.yml builds and deploys dist.

Vite base path is configured in vite.config.js as /Novltype/ for project-page hosting.

## PWA Notes

- Manifest is generated at build time.
- Service worker is generated by Workbox through vite-plugin-pwa.
- Icons are stored in public/icons.
- For updates on deployed builds, do a hard refresh if the old service worker is cached.
